# pcat-project
A RESTful API with a basic CRUD operations.

## Getting Started
Clone the repo:

```
git clone https://github.com/nisanurbaydas/pcat-project
cd pcat-project
```
Install the dependencies:

```
npm install
```
To start project:
```
npm start
```
